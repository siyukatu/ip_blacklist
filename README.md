# ip_blacklist
5分に1回自動的に更新されます。<br>
区切り文字は\nです。<br>
ご自由にお使いください。
## ブラックリスト登録の仕組みについて
大量のVPSやIPアドレスを所有し、sshやproxy等を探し回るbotからの接続を受け付けIPアドレスを収集しています。<br>
また、公開プロキシ等もこのリストに登録されています。
## ブラックリストからの削除について
削除には異議申し立てによる削除と自動削除の2種類があります。<br>
最終目撃から60日が経過すると、自動的に削除されます。<br>
自動削除の場合は削除後1年間、過去に登録されていたIPアドレスとしてサーバーに記録されます。
